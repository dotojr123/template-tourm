1.1 Requirements

There are system requirements in order to install and setup Tourm template and its components properly. Make sure that you are running the node.js and npm in your computer. You have to maintain this version (Node js >= 20.10.0 and npm >= 10.2.3)

Recommended Version Nodejs v20.10.0

If you don't have Node js in your computer please download and install from here: https://nodejs.org/en/download/

Install Packages

You can use Npm/Yarn

NPM : You must have to npm install packages. You can do this by running npm install from the root of your project to install all the necessary dependencies.

                            
                                npm install
Development server

Run npm run dev for a dev server. Navigate to http://localhost:3000/ . The app will automatically reload if you change any of the source files.

                            
                                npm start
